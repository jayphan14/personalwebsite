# Assets

## Profile photo

Save your LinkedIn photo here as `profile.jpg` (exact filename).
The `index.html` hero will pick it up automatically; the SVG monogram is a fallback.

To grab it from LinkedIn:
1. Open https://www.linkedin.com/in/jay-pphan/
2. Click your profile photo → "Download"
3. Save as `assets/profile.jpg`

A square-cropped image around 512×512 works well; CSS rounds it to a circle.
